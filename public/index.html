Search API Provider

🔑 How to get a free SerpApi key? (Beginner Friendly)
  1. Go to SerpApi.com and sign up for a free account.
  2. Verify your email to activate the account.
  3. Go to your Dashboard, find your private API Key, copy it, and paste it here!
  4. Free plans get **100 searches per month**, which is plenty for local exploration!

Razorpay Subscription Gate

Configure your API keys to enable live subscription checkout payments.

🔑 How to get your Razorpay Keys?
  1. Log in to your Razorpay Dashboard.
  2. Ensure you are in Test Mode (toggle in top header).
  3. Go to Account & Settings → API Keys.
  4. Click Generate Key to retrieve both Key ID and Key Secret.
